Browsers.

There once was a time when it made practical sense for web developers to bend over backwards in order to write code that formatted properly in Internet Explorer, Mozilla Firefox, and Safari in the same way. Luckily, many frameworks handle these quirks of individual browsers for you now. All you have to worry about is validating properly!
